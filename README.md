# Family Japan Trip 2026｜Tabelog 美食地圖版

此資料夾可直接上傳 GitHub Pages。

## 新增功能
- 餐廳依 DAY 1–DAY 8 自動分類
- 可依日期、料理類型與關鍵字篩選
- 餐廳卡片顯示 Tabelog 分數狀態、價位與推薦料理
- 每家餐廳附 Google 導航、Google 地圖及 Tabelog 搜尋
- Leaflet 互動餐廳地圖
- 手機橫向滑動卡片
- 已更新 11/5 CI108 航班與 20:30 銀座飯店家人會合資訊

## GitHub Pages 上傳
1. 將 `index.html` 上傳到 GitHub Repository 根目錄。
2. 進入 Settings → Pages。
3. Source 選 Deploy from a branch。
4. Branch 選 main、資料夾選 /root，按 Save。

## 注意
Tabelog 評分、營業時間與價位會變動。網站內標示「3.70+ 候選」的店家，請在出發前透過卡片中的 Tabelog 按鈕確認最新資料。
